# 

## How to 
- `bun run install:all` 
    - to install all dependencies
- `bun run dev`
  - to start server
      - frontend: port 3000
      - backend: port 8080

### Development 
- Target /api/ for backend
- Target / for frontend
- Uses proxy to auto route to backend if link starts with "/api"

## Personal use 
- Most likely using vercel for launching frontend 
- Render for launching backend
